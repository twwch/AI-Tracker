# 阿里 ATH 事业群发布首款 AI 开发工具 Meoo：集成四大模型，1 分钟生成全栈应用

4 月 15 日，阿里巴巴 ATH（Alibaba Token Hub）事业群正式发布旗下首款 AI 开发工具 **Meoo（秒悟）**，面向所有用户开放，无需邀请码即可使用。

![Meoo 产品界面](https://cdn.jsdelivr.net/gh/twwch/images/AI-Tracker/images/2026/04/meoo-interface.png)

## ATH 是什么？

今年 3 月 16 日，阿里宣布成立 **ATH 事业群**，与电商、云智能并列为集团核心业务板块，由 CEO 吴泳铭亲自挂帅。ATH 的战略定位是"创造 Token、分发 Token、应用 Token"，标志着阿里从"卖模型"转向"卖 Token/服务"的路线转型。ATH 下辖通义实验室、MaaS 平台、Qwen 事业部（C 端助手）、悟空事业部（B 端 AI）和 AI 创新事业部五大业务线。

## Meoo 核心能力

**集成四大顶尖模型：**

- Qwen3-Coder（阿里自研）
- Kimi K2.5（月之暗面）
- GLM-5（智谱 AI）
- MiniMax

**三种工作模式：**

| 模式 | 特点 |
|------|------|
| **Fast** | 快速生成，适合简单任务 |
| **Agent** | 自主规划、任务拆解，自动检测修复问题 |
| **Swarms** | 多 Agent 并行协作，最快 1 分钟出结果 |

**零代码全栈开发：** 用户无需任何编程基础，用自然语言描述想法，Meoo 即可生成包含前端 + 后端的完整网站或 H5 页面。

**一键部署阿里云：** 内置打通阿里云数据库、对象存储、域名服务、FC 沙箱、NAS 文件系统、百炼模型 API 等核心产品，无需用户手动配置。

**可视化页面编辑：** 用户可以圈选 UI 元素，用自然语言下达修改指令（如"字体放大"、"换个背景色"），AI 自动定位并修改代码。

**技能市场：** 支持模块化能力封装，用户可上传自定义技能或从市场下载，覆盖开发、测试、文档、数据、设计、效率、运维等类别，还支持调用平台 MCP 库扩展能力。

## 实测体验

新京报贝壳财经记者实测，约 **3 分钟**即可生成一个可用的待办计时 Web 应用。Meoo 在编码前会主动提出 4 个澄清问题，然后自动生成代码，遇到 bug 还能自行排查修复。

## 内部数据

- 阿里内部已有 **超 1 万名员工** 使用 Meoo
- 用户主要来自 **非技术岗位**：财务、设计、产品、运营
- 典型用例：财务团队自动生成报销管理系统，设计团队从文字描述生成交互原型

## 竞品定位

Meoo 对标 **Lovable** 和 **Bolt.new**，核心差异化优势：
- 原生中文优化
- 深度集成阿里云基础设施（无需海外服务）
- 简单应用分钟级部署
- 免费增值定价模式
